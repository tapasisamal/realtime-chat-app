# 💬 Real-Time Chat App (React + Supabase)

A real-time chat application built using React and Supabase.  
It allows users to sign up, log in, and send messages instantly in a shared global chat room with live updates.

---

## 🌐 Live Demo

🔗 [Click here to use Chat App]()

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Supabase (Auth + Database + Realtime)
- PostgreSQL (via Supabase)
- React Icons
- CSS

---

## ✨ Features

- User Authentication (Signup & Login)
- Real-time messaging with live updates
- Global chat room (all users can interact)
- Auto-scroll to latest messages
- Message timestamps
- Clean and responsive UI
- Supabase Realtime integration

---

## 📦 Installation & Setup

```bash
git clone <your-repo-link>
cd project-name
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in the root of the project and add the following:

```env
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```
